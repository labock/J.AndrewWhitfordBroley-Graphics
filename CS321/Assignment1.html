CSCI 321—Graphics

Programming Assignment 1: due Wednesday, January 27, 2016

The purpose of this assignment is to help you become familiar with the WebGL graphics environment using HTML and JavaScript, as well as to give you practice modeling 2-D objects. Note that the sample programs are in the directory /usr/people/classes/CS321/handouts/CourseSamples.

This is an individual programming assignment.

  1. Using the sample program object1 as a model, make three two-dimensional programs, each of which puts an object in the center of the canvas, using different object and background colors for each program. In each program, the object must consist of at least two triangles, and at least one of the objects must require three or more; at least one of the programs should display a non-convex object by breaking it into triangles. Here are some suggestions: a cyan diamond, a magenta regular pentagon, a yellow chevron (v-shape) on a black background, a five- or six-pointed star in pink or orange.
  2. Using the sample program object2 as a model, make non-curved versions of the four playing-card suits in the four quadrants of the canvas: hearts and diamonds in red, clubs and spades in black. Try using octagons or hexagons for the three balls in the club symbol; use parts of those shapes for the curved parts of hearts and spades.
